# Pinned runtime dependencies for reproducible deploys.
#
# These are the exact versions the project was built and tested against, so a
# deploy matches local. pyproject.toml holds the abstract spec (floors); this
# file is the concrete lock. Dev-only tools (pytest, ruff) are deliberately left
# out, since a deploy box shouldn't carry them. The Langfuse/OpenTelemetry stack
# is included so tracing can be switched on with env vars alone; it stays a no-op
# until LANGFUSE_PUBLIC_KEY is set.
#
# Deploy in two steps so the package itself is installed without re-resolving:
#   pip install -r requirements-lock.txt
#   pip install . --no-deps
# then start with:  uvicorn rpg.web:app --host 0.0.0.0 --port $PORT
#
# Regenerate after a deliberate upgrade with:  pip freeze (then prune dev tools).
